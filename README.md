# TChat

一个基于Java语言的多人聊天室项目，由终端版本出发改进，可长期改进项目。

## 多人多房间终端聊天1.0

### 实现需求
- 服务端
  - 显示聊天人数
  - 聊天房间创建显示
  - 用户登录连接显示
- 客户端
  - 创建退出房间
  - 房间内发言聊天
